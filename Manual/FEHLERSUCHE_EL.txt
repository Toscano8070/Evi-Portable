===========================================================================
  Evi Portable — Εκτεταμένη αντιμετώπιση προβλημάτων (αναφορά)
  Εγχειρίδιο συμπληρωμάτων\README.txt (επισκόπηση) και μοντέλα\llm\README.txt
===========================================================================

Αυτός ο οδηγός καλύπτει τυπικά και λιγότερο κοινά προβλήματα: αιτία, έλεγχοι, διορθώσεις.
Εργαστείτε από πάνω προς τα κάτω ή μεταβείτε στο αντίστοιχο τμήμα.

Σημείωση: Δεν αναφέρεται κάθε μήνυμα στην οθόνη αυτολεξεί εδώ — χρησιμοποιήστε τον συνδυασμό
του συμπτώματος (τι βλέπετε;) και της επικεφαλίδας της ενότητας.

------------------------------------------------------------------------------
  Γρήγορος έλεγχος (πάντα πρώτος)
------------------------------------------------------------------------------

  □ Windows 10 ή 11 (64-bit), τρέχουσες ενημερώσεις, επανεκκίνηση μετά από σημαντικές αλλαγές;
  □ Η Evi ξεκίνησε από έναν πλήρως εξαγόμενο φάκελο — όχι από ένα μισό αποσυμπιεσμένο αρχείο,
    και όχι "live" μόνο από το cloud (φάκελος συγχρονισμού);
  □ Τουλάχιστον ένα έγκυρο .gguf στα μοντέλα\llm\ (δείτε μοντέλα\llm\README.txt);
  □ Αρκετός ελεύθερος χώρος στο δίσκο (μοντέλο + συνομιλίες + ενημερώσεις);
  □ Μετά από αλλαγές σε ρυθμίσεις, φύλλα εξαπάτησης ή μοντέλα: κλείστε πλήρως την Evi
    και να ξαναρχίσω;


------------------------------------------------------------------------------
  1. Βασικά στοιχεία του λειτουργικού συστήματος
------------------------------------------------------------------------------

Σύμπτωμα: Σφάλματα εγκατάστασης ή εκκίνησης, "δεν υποστηρίζεται σε ..."
Εξήγηση: Η Evi στοχεύει μόνο τα Windows 10/11. Άλλα συστήματα δεν υποστηρίζονται.

Διορθώσεις:
  • Χρησιμοποιήστε μόνο Windows 10 ή 11.
  • Τα Windows 32 bit συνήθως δεν υποστηρίζονται — χρησιμοποιήστε 64 bit.
  • Εγκαταστήστε ενημερώσεις των Windows, επανεκκινήστε τον υπολογιστή.

Σύμπτωμα: Το διπλό κλικ στην εφαρμογή δεν κάνει τίποτα
Επεξήγηση: Αποκλεισμός πολιτικής, λείπουν δικαιώματα ή πρόγραμμα εκκίνησης σε αποκλεισμένη διαδρομή.

Διορθώσεις:
  • Κάντε δεξί κλικ → Εκτέλεση ως διαχειριστής μόνο εάν χρειάζεται (κανονική περίπτωση: τυπικός χρήστης).
  • Μην εκτελείτε μέσα από ένα ZIP — εξάγετε πλήρως.
  • Ελέγξτε εάν το πρόγραμμα προστασίας από ιούς έχει καταγράψει ένα μπλοκ (βλ. ενότητα 3).


------------------------------------------------------------------------------
  2. Διαδρομές φακέλων, φορητότητα, cloud & antivirus
------------------------------------------------------------------------------

Σύμπτωμα: Τα αρχεία "εξαφανίζονται", διπλοί φάκελοι, κατεστραμμένο .gguf
Επεξήγηση: Ο συγχρονισμός στο cloud (OneDrive, Dropbox,…) μπορεί να κλειδώσει αρχεία ή να ανεβάσει μερική
  δεδομένα ενώ η Εύη γράφει.

Διορθώσεις:
  • Τοποθετήστε το φάκελο Evi σε μια τοπική μονάδα δίσκου, π.χ. C:\EviPortable ή D:\Tools\Evi —
    όχι "Documents\OneDrive\..." με ενεργό συγχρονισμό στα ζωντανά δεδομένα.
  • Προσθέστε μια εξαίρεση λογισμικού ασφαλείας για τη ρίζα Evi (σαρώνοντας κάθε μικρό
    Η εγγραφή μπορεί να επιβραδύνει ή να κλειδώνει άσχημα το LLM/Piper).

Σύμπτωμα: Εγκατάσταση USB — πολύ αργή ή σφάλματα
Εξήγηση: Slow sticks, όρια μεγέθους FAT32, κακή σύνδεση.

Διορθώσεις:
  • USB 3.x stick, NTFS, αρκετός χώρος. προτιμήστε εσωτερικό SSD.
  • Μην εκτελείτε τεράστια μοντέλα σε πολύ αργά μέσα — χρησιμοποιήστε μικρότερο GGUF.

Σύμπτωμα: «Πολύ μεγάλη διαδρομή» ή σφάλματα ειδικών χαρακτήρων
Επεξήγηση: Windows MAX_PATH / Unicode — σπάνιες, αλλά βαθιές διαδρομές + περιττοί χαρακτήρες
  σπάστε μερικά εργαλεία.

Διορθώσεις:
  • Χρησιμοποιήστε μια σύντομη διαδρομή, π.χ. C:\Evi\, αποφύγετε τη διαδρομή που δεν είναι ASCII εάν είναι δυνατόν.


------------------------------------------------------------------------------
  3. Μοντέλο γλώσσας (.gguf) — δεν υπάρχουν απαντήσεις, σφάλματα μοντέλου, κολλάει
------------------------------------------------------------------------------

Σύμπτωμα: "Χωρίς AI", κενές απαντήσεις, το μοντέλο δεν θα φορτωθεί
Επεξήγηση: Δεν υπάρχει έγκυρο GGUF στον αναμενόμενο φάκελο, ημιτελής λήψη, λάθος
  μορφή ή κατεστραμμένο αρχείο.

Διορθώσεις:
  • Ανοιχτά μοντέλα\llm\: τουλάχιστον ένα .gguf με εύλογο μέγεθος (όχι 0 byte).
  • Επανάληψη λήψης. στο πρόγραμμα περιήγησης ελέγξτε για μονές επεκτάσεις στο όνομα — μετονομασία
    μόνο αν γνωρίζετε ότι το αρχείο είναι πλήρες.
  • Ανάγνωση μοντέλων\llm\README.txt: προτεινόμενα αρχεία Qwen-2.5 και επίπεδα VRAM.
  • Πολλαπλά GGUF: ανά README μπορεί να υπάρχουν πολλά αρχεία — η Evi επιλέγει να ταιριάζει με την VRAM.
    Εάν έχετε αμφιβολίες, αφήστε μόνο ΕΝΑ γνωστό-καλό αρχείο στο φάκελο.

Σύμπτωμα: Απαντάει εξαιρετικά αργά παρά τη GPU
Επεξήγηση: Το μοντέλο είναι πολύ μεγάλο για GPU → offload/CPU ή λάθος προεπιλογή.

Διορθώσεις:
  • Μικρότερο μοντέλο (π.χ. 7B q4_k_m αντί για 14B).
  • Αντιστοιχίστε την προκαθορισμένη GPU στην πλαϊνή γραμμή με το πραγματικό υλικό.
Σύμπτωμα: Σφάλμα CUDA / GPU, "εξαντλήθηκε η μνήμη", σφάλμα κατά τη δημιουργία
Εξήγηση: Δεν αρκεί η VRAM, μια άλλη διαδικασία χρησιμοποιεί τη GPU, ο φορητός υπολογιστής έχει κολλήσει στο iGPU.

Διορθώσεις:
  • Ενημερώστε το πρόγραμμα οδήγησης NVIDIA (nvidia.com).
  • Σε φορητούς υπολογιστές, στο BIOS/Windows επιβάλλουν «διακριτή GPU» / «υψηλές επιδόσεις» για την Evi
    όπου είναι διαθέσιμο.
  • Δοκιμάστε την προεπιλογή CPU στο Evi (αργή αλλά σταθερή).
  • Κλείστε άλλες εφαρμογές που απαιτούν GPU (παιχνίδια, άλλα εργαλεία τεχνητής νοημοσύνης).


------------------------------------------------------------------------------
  4. GPU, προγράμματα οδήγησης, υβριδικά γραφικά φορητού υπολογιστή
------------------------------------------------------------------------------

Σύμπτωμα: Η Evi δεν βλέπει NVIDIA GPU παρόλο που είναι εγκατεστημένη μία
Εξήγηση: Λανθασμένη λειτουργία τροφοδοσίας, λείπει το πρόγραμμα οδήγησης, υβριδικό πάντα σε iGPU.

Διορθώσεις:
  • Διαχείριση Συσκευών: NVIDIA χωρίς κωδικό σφάλματος;
  • Λειτουργία "Υψηλή απόδοση" / "Απόλυτη" (εφαρμογές προμηθευτή: Lenovo Vantage,…).
  • Επανεκκίνηση μετά την ενημέρωση του προγράμματος οδήγησης.

Σύμπτωμα: Τρεμοπαίξιμο οθόνης, συντριβή προγράμματος οδήγησης (TDR)
Εξήγηση: Λήξη χρονικού ορίου γραφικών — συχνά υπερβολικό φορτίο GPU ή ασταθές overclock.

Διορθώσεις:
  • Χωρίς υπερχρονισμό κατά τη διάρκεια των δοκιμών. μικρότερο μοντέλο? καλύτερη ψύξη.


------------------------------------------------------------------------------
  5. Μόνο CPU — προσδοκίες και όρια
------------------------------------------------------------------------------

Σύμπτωμα: «Χρειάζεται για πάντα»
Εξήγηση: Το συμπέρασμα CPU στο 7B+ είναι πολύ πιο αργό από το GPU — αυτό είναι φυσιολογικό.

Διορθώσεις:
  • Διατηρήστε αρκετή μνήμη RAM ανά μοντέλα\llm\README.txt.
  • Κλείσιμο εργασιών στο παρασκήνιο. σχέδιο ισχύος «Υψηλή απόδοση».


------------------------------------------------------------------------------
  6. Ενεργοποίηση, άδεια, Μηχάνημα ID
------------------------------------------------------------------------------

Σύμπτωμα: Η σειρά δεν γίνεται αποδεκτή
Επεξήγηση: Τυπογραφικά λάθη, κενά, κλειδί για διαφορετικό υλικό, παλιό αναγνωριστικό.

Διορθώσεις:
  • Επικολλήστε προσεκτικά το κλειδί, χωρίς κενά προπορευόμενα/υστερούντα.
  • Βεβαιωθείτε ότι η αλληλογραφία ταιριάζει με αυτόν τον υπολογιστή / έγκυρο αναγνωριστικό μηχανήματος.
  • Μετά την αλλαγή mainboard/CPU: νέο ID — επικοινωνήστε με την υποστήριξη με proof chain.

Σύμπτωμα: Δεν υπάρχει απάντηση αλληλογραφίας
Επεξήγηση: Φάκελος ανεπιθύμητης αλληλογραφίας, λάθος Από διεύθυνση, λάθος θέμα.

Διορθώσεις:
  • Έλεγχος ανεπιθύμητης αλληλογραφίας. στείλτε ξανά με σαφές θέμα και αναγνωριστικό μηχανήματος.


------------------------------------------------------------------------------
  7. Μικρόφωνο — χωρίς επίπεδο, η Εύη δεν ακούει τίποτα
------------------------------------------------------------------------------

Σύμπτωμα: Καμία σύλληψη, μετρητής κενός
Επεξήγηση: Τα Windows αποκλείουν την πρόσβαση, λάθος συσκευή εισόδου, καλώδιο/USB.

Διορθώσεις:
  • Ρυθμίσεις → Απόρρητο → Μικρόφωνο → επιτρέπονται οι εφαρμογές επιφάνειας εργασίας.
  • Ρυθμίσεις ήχου → Είσοδος: ορίστε και δοκιμάστε τη σωστή προεπιλογή ("ομιλία").
  • Στο Evi: επιλέξτε μικρόφωνο, Αποθήκευση, επανεκκίνηση εφαρμογής.
  • Ακουστικά Bluetooth: σύζευξη, μπαταρία. Η καθυστέρηση μπορεί να είναι κακή — δοκιμάστε το μικρόφωνο USB.

Σύμπτωμα: Το μικρόφωνο λειτουργεί μόνο σε ορισμένες εφαρμογές
Εξήγηση: Η αποκλειστική λειτουργία άλλης εφαρμογής αποκλείει τη συσκευή.

Διορθώσεις:
  • Κλείστε άλλες εφαρμογές επικοινωνίας ή απενεργοποιήστε το "αποκλειστικό μικρόφωνο".


------------------------------------------------------------------------------
  8. Αναγνώριση ομιλίας (τοπική)
------------------------------------------------------------------------------

Σύμπτωμα: Η ομιλία δεν αναγνωρίζεται, «κολλημένη» επεξεργασία
Επεξήγηση: Λείπει/μη έγκυρο μοντέλο Whisper, λήψη για πρώτη φορά, κατειλημμένη CPU/GPU.

Διορθώσεις:
  • Δίκτυο μόνο εάν πρέπει να γίνει ανάκτηση μοντέλων — βλ. μοντέλα\whisper.
  • Πληκτρολογήστε σύντομα αντί να μιλήσετε για να απομονώσετε την Εύη.
  • Μετά από σημαντικές ενημερώσεις: επανεκκινήστε την Evi.


------------------------------------------------------------------------------
  9. Έξοδος φωνής (TTS / Piper) — σιωπηλή ή λάθος γλώσσα
------------------------------------------------------------------------------

Σύμπτωμα: Δεν υπάρχει ήχος αν και εμφανίζεται η απάντηση κειμένου
Επεξήγηση: Δεν υπάρχει πακέτο φωνής για τη γλώσσα, λάθος καταχώρηση φωνής, ένταση των Windows.

Διορθώσεις:
  • Έλεγχος piper\ — φωνές για την επιλεγμένη γλώσσα που υπάρχει;
  • Στη γλώσσα διεπαφής χρήστη Evi pick φωνής, αποθήκευση.
  • Μείκτης έντασης των Windows: Η Evi δεν είναι σε σίγαση;
  • Δοκιμάστε μια άλλη συσκευή εξόδου (ακουστικά έναντι οθόνης).


------------------------------------------------------------------------------
 10. Δίκτυο, τείχος προστασίας, "Αποκλεισμός όλης της κυκλοφορίας"
------------------------------------------------------------------------------

Σύμπτωμα: Ιστός, βοηθοί αλληλογραφίας, καιρός, λήψεις αποτυγχάνουν
Επεξήγηση: Καθολικό μπλοκ στο Evi, τείχος προστασίας Windows, εταιρικός διακομιστής μεσολάβησης, VPN.

Διορθώσεις:
  • Στο Evi απενεργοποιήστε την επιλογή "Αποκλεισμός όλης της κυκλοφορίας" (ή ισοδύναμο) εάν διαθέτει σύνδεση
    αναζητούνται.
  • Χαλαρώστε προσωρινά το VPN/το τείχος προστασίας (μόνο εάν επιτρέπεται).
  • Εταιρικό δίκτυο: ζητήστε από την IT εξαιρέσεις διακομιστή μεσολάβησης.


------------------------------------------------------------------------------
 11. YouTube, media, VLC Portable
------------------------------------------------------------------------------

Σύμπτωμα: Το βίντεο δεν ξεκινά, "λείπει στοιχείο"
Επεξήγηση: Λείπει το φορητό VLC, έχει αποκλειστεί το δίκτυο, λάθος διαδρομή.

Διορθώσεις:
  • Μην διαγράψετε το Portable\VLCPortable (ή την παρεχόμενη διάταξη).
  • Να επιτρέπεται το δίκτυο (βλ. ενότητα 10).


------------------------------------------------------------------------------
 12. Γλώσσα διεπαφής χρήστη, cheat sheets
------------------------------------------------------------------------------

Σύμπτωμα: Η εντολή δεν λειτουργεί, παλιό κείμενο
Επεξήγηση: Το φύλλο εξαπάτησης επεξεργάστηκε αλλά η Εύη δεν επανεκκίνησε. λάθος όνομα αρχείου.

Διορθώσεις:
  • Ανάγνωση cheatsheets\README.txt — ένα αρχείο ανά κωδικό γλώσσας (π.χ. de.txt, en.txt).
  • Μετά από χειροκίνητες επεξεργασίες επανεκκινήστε πλήρως την Evi.
  • Αποθήκευση αρχείου ως UTF-8 (τονισμοί).


------------------------------------------------------------------------------
 13. Ασφάλεια: PIN και φωνητικό ξεκλείδωμα
------------------------------------------------------------------------------

Σύμπτωμα: Ξέχασα το PIN, το φωνητικό ξεκλείδωμα αποτυγχάνει πάντα
Εξήγηση: Λειτουργία ασφαλείας — σκόπιμα δύσκολο να παρακαμφθεί. δωμάτιο πολύ θορυβώδες.

Διορθώσεις:
  • Ήσυχο δωμάτιο, επίπεδο μικροφώνου, χωρίς επικαλυπτόμενες φωνές από άλλους.
  • PIN/προσωπική ανάκτηση μόνο ανά πολιτική προμηθευτή — υποστήριξη με απόδειξη.
  • Μην κοινοποιείτε ποτέ το PIN σας.


------------------------------------------------------------------------------
 14. Απόδοση, κολλήματα, κούτσουρα
------------------------------------------------------------------------------

Σύμπτωμα: Πάγωμα, ξαφνική συντριβή
Επεξήγηση: VRAM, RAM, θερμικό γκάζι, κακό αρχείο, σφάλμα μετά την ενημέρωση.

Διορθώσεις:
  • Μικρότερο μοντέλο. κλείστε άλλες εφαρμογές. ελέγξτε την ψύξη.
  • Ελέγξτε το crash.log ή κάτι παρόμοιο στο φάκελο Evi εάν υπάρχει.
  • Αναίρεση τελευταίας αλλαγής (κατάργηση νέου μοντέλου, επαναφορά της ρύθμισης).

Σύμπτωμα: Όλα φαίνονται νωχελικά
Επεξήγηση: Υπηρεσίες παρασκηνίου, πλήρης SSD, πολύ μεγάλο πλαίσιο μοντέλου.

Διορθώσεις:
  • Διαχείριση εργασιών: CPU/RAM/GPU. σχέδιο ενέργειας? ελεύθερος χώρος SSD (περίπου 10–15% κενός).


------------------------------------------------------------------------------
 15. Εάν τίποτα δεν βοηθάει — πληροφορίες για την αλληλογραφία υποστήριξης
------------------------------------------------------------------------------

  • Αναγνωριστικό μηχανήματος (ενεργοποίηση / οθόνη πληροφοριών)
  • Έκδοση Windows (winver), 64-bit
  • Μοντέλο GPU + VRAM + έκδοση προγράμματος οδήγησης
  • Ποιο .gguf (όνομα αρχείου, κατά προσέγγιση μέγεθος)
  • Σύντομα βήματα που κάνατε
  • Ακριβές κείμενο σφάλματος ή στιγμιότυπο οθόνης (μην εμφανίζονται τα ιδιωτικά κλειδιά!)
