  문서: 영어 • 이 폴더와 함께 받은 빌드에 적용됩니다.

자신만의 긴 가이드나 번역도 여기에 넣으세요(예: EN_User_Manual.pdf,
DE_Handbuch.txt). 이 파일은 Evi를 실행하기 위한 선택 사항입니다. 그건 당신이 읽어야 할 일이에요
또는 인쇄.

현지화된 복사본: 이 폴더에 있는 README_<LANG>.txt 및 FEHLESUCHE_<LANG>.txt
(웹사이트 UI와 동일한 언어 코드) 영어 개요: 이 README.txt;
영어 문제 해결: FEHLERSUCHE_EN.txt. 독일어 문제 해결도
FEHERSUCHE_DE.txt로 직접 관리됩니다.

폴더 색인(파일 이름 지정, 생성기): README.md

--------------------------------------------------------------------------------
  운영 체제
--------------------------------------------------------------------------------

Evi Portable은 Microsoft Windows 10 및 Windows 11 전용입니다. 그렇지 않다
Windows 7/8, macOS, Linux 또는 모바일 운영 체제에서 지원됩니다.

--------------------------------------------------------------------------------
  첫 번째 시작 - 다운로드부터 Evi 실행까지
--------------------------------------------------------------------------------

1) 설치 위치
   예를 들어 Evi 폴더 전체를 제어할 수 있는 위치에 복사하거나 추출합니다.
   데스크탑, 문서 또는 USB 스틱. 폴더 구조를 다음과 같이 유지하십시오.
   전달됩니다(직접 생성하지 않은 하위 폴더는 삭제하지 마십시오).

2) 언어모델(필수)
   Evi에는 폴더에 하나의 기본 AI 모델 파일(확장자 .gguf)이 필요합니다.
   모델\llm\
   크기 권장 사항은 models\llm\README.txt를 참조하세요. 파일이 거기에 있는 후에,
   Evi를 시작하십시오; 가능하면 모델을 자동으로 감지합니다.

3) 첫 번째 실행 - 활성화
   Evi를 처음 시작하면 활성화 창이 나타납니다.
   • 귀하의 PC에는 컴퓨터 ID(이 하드웨어에 고유함)가 표시됩니다.
   • 컴퓨터 ID를 복사하고(버튼: "컴퓨터 ID 복사") 이메일로 다음 주소로 보냅니다.
     Brielbeck@hotmail.de
   • 동일한 PC에서만 작동하는 시리얼 키를 받게 됩니다.
     (키는 하드웨어에 바인딩되어 있습니다).
   • 활성화 창에 키를 붙여넣고 활성화를 선택합니다.

   주요 하드웨어를 변경하거나 ID를 변경하는 방식으로 디스크를 이동하는 경우,
   새 키가 필요할 수 있습니다. 동일한 주소로 문의하세요.

4) 활성화 후
   메인 창이 열립니다. 왼쪽에서 언어, 마이크, 음성을 선택하세요.
   설정 패널. 명령 예시를 보려면 언제든지 "치트 시트 열기"라고 말하세요.
   (다른 언어에 대해서는 치트시트\en.txt 및 자매 파일을 참조하세요).

5) 선택 사항: Windows에서 이 설명서를 엽니다.
   파일 탐색기에서 Evi 폴더로 이동한 다음 Manual\README.txt로 이동하여 엽니다.
   메모장이나 텍스트 뷰어를 사용하세요.

--------------------------------------------------------------------------------
  인터페이스 언어(GUI)
--------------------------------------------------------------------------------

사용자 인터페이스는 상단의 언어에서 선택한 언어를 따릅니다.
왼쪽 설정 패널(사이드바의 첫 번째 블록) 다음을 선택한 후
언어, 메뉴, 버튼, 라벨 및 대부분의 화면 텍스트가 해당 언어로 전환됩니다.
번역이 존재하는 모든 언어. 언제든지 변경할 수 있습니다. 오직
레이아웃이 아니라 문구가 변경됩니다.

팁: cheatsheets\ 폴더의 명령 치트 시트는 다음과 같이 나열됩니다.
언어 코드(en.txt, de.txt, …). 자세한 내용은 치트시트\README.txt를 참조하세요.
이 Manual\ 폴더는 더 긴 제품 개요입니다. 치트 시트는
빠른 "무슨 말" 목록.

--------------------------------------------------------------------------------
  그래픽 카드(GPU) - 퀵 가이드
--------------------------------------------------------------------------------

강력한 중간 선택은 12GB 비디오를 갖춘 NVIDIA GeForce RTX 4070 Ti입니다.
메모리: 속도, 컨텍스트 크기 및 더 큰 모델을 위한 공간의 균형이 잘 맞습니다.

약 RTX 3060부터 RTX 5090까지의 카드가 잘 작동할 수 있습니다. 가장 잘 맞는 것은 달려있다
RAM, CPU, 냉각 및 설치하는 모델 파일에 대해 설명합니다. GPU 사전 설정 사용
귀하의 하드웨어와 일치하는 Evi의 사이드바.

CPU에서만 실행("CPU … GB RAM" 사전 설정)은 긴급 폴백입니다.
Evi는 적절한 그래픽 카드 없이도 계속 사용할 수 있지만 답변은 훨씬 느립니다.
가능하다면 실제 GPU를 선호하십시오.

--------------------------------------------------------------------------------
  완벽한 휴대성 — 인터넷이 필요할 때
--------------------------------------------------------------------------------

Evi는 휴대 가능하도록 제작되었습니다. 전체 폴더를 다른 드라이브, USB 스틱에 복사하고,
또는 PC를 선택하고 거기서 시작하세요. 채팅, 추억, 설정, 보안 데이터 및
활성화 파일은 일반적으로 해당 폴더 안에 있습니다(활성화는
PC, 폴더 경로 아님).

인터넷은 주로 다운로드하기로 선택한 항목(모델 파일, 선택 사항)을 위한 것입니다.
음성 또는 추가 기능, 업데이트) 및 웹 검색, 메일,
네트워크 액세스를 허용할 때 스트리밍 또는 날씨.

허용된 폴더 내부의 핵심 채팅, 메모리, 메모, 타이머, 파일 도구, 로컬
음성 인식, 로컬 음성 출력 및 개인 정보 보호 잠금이 실행됩니다.
대화를 클라우드 서비스로 보내지 않고도 컴퓨터에 보낼 수 있습니다. 온라인 선택 사항
기능은 네트워크 액세스가 켜져 있고 사용할 때만 실행됩니다.

--------------------------------------------------------------------------------
  보안: PIN 및 음성 잠금 해제(복사 방지)
--------------------------------------------------------------------------------

Evi는 장치에서 보안 PIN과 선택적 음성 등록을 사용할 수 있습니다.

음성 잠금 해제를 사용하면 시도할 때마다 짧은 무작위 단어 세트를 요청할 수 있습니다.
매번 바뀌는 것입니다. 영원히 고정된 문구가 하나도 아닙니다. 그건 쉬운 일을 막는다
당신의 목소리가 녹음된 오래된 것을 다시 재생하는 방법입니다. 귀하의 PIN은
별도의 방어선.

이는 우발적인 학대를 막기 위해 고안되었습니다. 어떤 소비자 제품도 약속할 수 없습니다
모든 공격에 대한 완벽함(예: PIN과 비밀번호를 모두 알고 있는 사람)
정교한 음성 모방). PIN을 비공개로 유지하고 설정을 완료하세요.
앱이 당신을 안내합니다.

--------------------------------------------------------------------------------
  문제 해결(간단한 체크리스트)
--------------------------------------------------------------------------------

• AI 응답 없음/모델 오류
  → models\llm\에 적합한 .gguf가 하나 이상 있습니까? models\llm\README.txt를 참조하세요.
  → GPU 경로가 실패하는 경우 더 작은 모델이나 사이드바의 CPU 사전 설정을 사용해 보십시오.

• 마이크 없음
  → Windows 사운드 설정: 마이크를 테스트합니다. Evi에서 아래의 장치를 선택하세요.
    마이크 및 저장.

• 음성 출력 없음
  → Piper\ 폴더에 해당 언어의 음성 파일이 있는지 확인하고
    사이드바에서 음성을 선택하세요.

• 웹이나 YouTube가 실패함
  → 온라인 기능을 원하시면 사이드바에서 "모든 트래픽 차단"을 끄세요.
  → YouTube 재생을 위해서는 일반적으로 VLC Portable 및 네트워크 액세스가 필요합니다.

• 치트시트 편집 후
  → Evi를 다시 시작하면 모든 텍스트 도우미가 변경 사항을 안정적으로 선택할 수 있습니다.

--------------------------------------------------------------------------------
  문제 해결(확장 참조)
--------------------------------------------------------------------------------

빠른 확인(항상 먼저)
  • Windows 10 또는 11 64비트, 업데이트, 큰 변경 후 재부팅?
  • Evi는 아카이브 내부가 아닌 완전히 추출된 폴더에서 실행됩니까?
  • 라이브 데이터 폴더에는 클라우드 동기화 폴더(OneDrive 등)를 사용하지 마세요.
    Evi 실행 — 다음과 같은 경우 C:\EviPortable 또는 D:\Tools\Evi와 같은 로컬 경로를 사용합니다.
    가능합니다.
  • 모델, 채팅, 업데이트를 위한 충분한 여유 디스크 공간이 있습니까?
  • 모델, 음성 또는 치트 시트를 변경한 후: Evi를 완전히 종료하고 시작합니다.
    다시.

1) 운영체제
  • Evi는 Windows 10 및 11 전용입니다. 다른 OS 버전은 지원되지 않습니다.
  • 앱이 전혀 시작되지 않으면 패키지를 완전히 추출하고 바이러스 백신을 확인하세요.
    차단된 실행 파일에 대한 로그를 기록하고 드물게 발생하지 않는 짧은 설치 경로를 시도합니다.
    유니코드 전용 문자입니다.

2) 폴더 경로, 바이러스 백신, 이식성
  • 검사 결과 다음과 같은 경우 보안 소프트웨어에서 Evi 루트 폴더에 대한 제외를 추가하세요.
    다운로드 중에 시작이 매우 느리거나 파일이 잠깁니다.
  • USB 스틱: USB 3 + NTFS를 선호합니다. 매우 느린 미디어는 대형 모델을 고통스럽게 만듭니다.

3) 언어 모델(.gguf)
  • 증상: 응답 없음, 모델 오류, 즉각적인 로드 실패.
  • 수정 사항: models\llm\에 완전한 .gguf(0바이트 아님)가 포함되어 있는지 확인합니다. 다시 다운로드
필요한 경우; models\llm\README.txt를 사용하여 모델 크기를 VRAM에 일치시킵니다. 확실하지 않은 경우
    테스트된 단일 q4_k_m 파일을 폴더에 보관하세요.
  • 여러 .gguf 파일: Evi는 VRAM에 맞는 가장 큰 파일을 선택할 수 있습니다.
    충돌이 의심되면 하나의 파일로만 테스트하세요.

4) GPU, CUDA, 드라이버, 메모리 부족
  • NVIDIA 드라이버를 업데이트하세요. 노트북에서는 개별 GPU/성능을 강제합니다.
    Windows에서 허용하는 Evi 모드입니다.
  • OOM 또는 GPU 충돌이 발생하는 경우 더 작은 체크포인트를 사용하고 다른 GPU 앱을 닫은 다음
    온도를 낮추거나 CPU 사전 설정(느리지만 안정적)으로 전환하세요.

5) CPU 전용 모드
  • 속도가 느릴 것으로 예상됩니다. 충분한 시스템 RAM을 확보하세요. 무거운 백그라운드 작업을 닫습니다.
    긴 세션 중에는 전원 계획 "고성능"을 사용하십시오.

6) 활성화
  • 키를 조심스럽게 붙여넣으세요(추가 공백 없음). 키는 하나의 컴퓨터 ID에 연결되어 있습니다.
  • 주요 하드웨어 변경 후에는 교체 키가 필요할 수 있습니다. 지원 센터를 이용하세요.
    귀하의 세부 사항을 주소로 보내십시오.

7) 마이크
  • Windows 개인정보 보호 → 마이크 → 데스크톱 앱을 허용합니다.
  • 올바른 기본 녹음 장치를 설정하십시오. Evi에서 동일한 장치를 선택하고 저장합니다.
  • Bluetooth 헤드셋으로 인해 대기 시간이 추가될 수 있습니다. 확실하지 않은 경우 USB 마이크를 사용하여 테스트해 보세요.
  • 다른 앱에는 독점 모드가 있을 수 있습니다. 회의 소프트웨어를 일시적으로 닫습니다.

8) 음성인식(로컬)
  • 인식이 완료되지 않으면 모델\속삭임 아래의 모델 파일이 있는지 확인하세요.
    그대로 유지하고 빌드에 필요한 경우 최초 모델 가져오기 시 네트워크를 허용합니다.

9) 음성 출력(파이퍼/TTS)
  • Piper\에 UI 언어용 음성 팩이 포함되어 있는지 확인하세요. 목소리를 고르다
    사이드바; Windows 볼륨 믹서를 확인하세요. Evi는 앱별로 음소거될 수 있습니다.

10) 네트워크 및 "모든 트래픽 차단"
  • 웹, 메일 도우미, 날씨 또는 다운로드에 대한 차단을 해제합니다.
  • 기업 프록시 또는 VPN의 경우 허용 목록을 작성하기 위해 IT 지원이 필요할 수 있습니다.

11) 유튜브와 미디어
  • 번들로 제공되는 VLC Portable 레이아웃을 유지합니다. 네트워크 액세스를 활성화합니다. 읽다
    경로가 이동된 경우 Portable\VLCPortable 지침입니다.

12) UI 언어 및 치트 시트
  • 치트 시트는 치트 시트\ 아래에 언어별로 파일이 있습니다. 치트 시트\README.txt를 참조하세요.
  • 편집 내용을 UTF-8로 저장합니다. 편집 후 Evi를 다시 시작하십시오.

13) 보안(PIN/음성 잠금해제)
  • 조용한 환경을 이용하십시오. 마이크 게인을 다시 확인하세요. PIN을 절대 공유하지 마세요.
  • 임의의 단어 프롬프트는 의도적인 것입니다. 오래된 음성 녹음은 잠금 해제되어서는 안 됩니다.

14) 성능, 정지, 충돌
  • 모델 크기를 줄이고, 냉각을 개선하고, 필요한 경우 트리에서 모든 crash.log를 읽습니다.
    build는 하나를 생성하고 실패하기 전에 수행한 마지막 변경 사항을 롤백합니다.

15) 지원팀에 문의할 때
  • 컴퓨터 ID, Windows 버전, GPU 모델 + VRAM + 드라이버(.gguf)를 포함합니다.
    사용하는 정보와 정확한 오류 텍스트(비밀 키 아님).
