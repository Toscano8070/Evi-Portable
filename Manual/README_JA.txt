--------------------------------------------------------------------------------
  オペレーティングシステム
--------------------------------------------------------------------------------

Evi Portable は Microsoft Windows 10 および Windows 11 のみに対応しています。そうではありません
Windows 7/8、macOS、Linux、またはモバイル オペレーティング システムでサポートされています。

--------------------------------------------------------------------------------
  最初のスタート - Evi のダウンロードから実行まで
--------------------------------------------------------------------------------

1) 設置場所
   Evi フォルダー全体を、制御する場所にコピーまたは抽出します。たとえば、
   デスクトップ、ドキュメント、または USB スティック。フォルダー構造はそのままにしておきます
   配信されました (自分で作成したものではないサブフォルダーは削除しないでください)。

2) 言語モデル (必須)
   Evi には、フォルダー内にメイン AI モデル ファイル (拡張子 .gguf) が 1 つ必要です。
   モデル\llm\
   推奨サイズについては、models\llm\README.txt を参照してください。ファイルがそこにある後、
   Eviを起動します。可能な場合はモデルを自動的に検出します。

3) 最初の起動 - アクティベーション
   Evi を初めて起動すると、アクティベーション ウィンドウが表示されます。
   • PC にはマシン ID (このハードウェアに固有) が表示されます。
   • マシン ID をコピーし (ボタン: [マシン ID をコピー])、電子メールで次の宛先に送信します。
     Brielbeck@hotmail.de
   • 同じ PC 上でのみ機能するシリアル キーを受け取ります。
     (キーはハードウェアにバインドされています)。
   • キーをアクティベーション ウィンドウに貼り付け、[アクティベート] を選択します。

   主要なハードウェアを変更したり、ID が変更されるような方法でディスクを移動した場合、
   新しいキーが必要になる場合がありますので、同じアドレスにお問い合わせください。

4) アクティベーション後
   メインウィンドウが開きます。左側で言語、マイク、音声を選択します
   設定パネル。コマンドの例については、いつでも「チートシートを開いて」と言ってください
   (他の言語については、cheatsheets\en.txt および姉妹ファイルを参照してください)。

5) オプション: このマニュアルを Windows から開きます
   ファイル エクスプローラーで Evi フォルダーに移動し、Manual\README.txt を開きます。
   メモ帳または任意のテキスト ビューアを使用して。

--------------------------------------------------------------------------------
  インターフェース言語（GUI）
--------------------------------------------------------------------------------

ユーザー インターフェイスは、上部の [言語] で選択した言語に従います。
左側の設定パネル (サイドバーの最初のブロック)。を選択した後、
言語、メニュー、ボタン、ラベル、およびほとんどの画面上のテキストがその言語に切り替わります。
翻訳が存在する言語であればどこでも使用できます。いつでも変更できます。だけ
レイアウトではなく、文言の変更です。

ヒント: cheatsheets\ フォルダー内のコマンド チート シートは、次のようにリストされています。
言語コード (en.txt、de.txt、…)。詳細については、cheatsheets\README.txt を参照してください。
この Manual\ フォルダーには、より長い製品概要が記載されています。カンニングペーパーは
簡単な「言いたいこと」リスト。

--------------------------------------------------------------------------------
  グラフィックス カード (GPU) — クイック ガイド
--------------------------------------------------------------------------------

有力な中間の選択肢は、12 GB のビデオを搭載した NVIDIA GeForce RTX 4070 Ti です。
メモリ: 速度、コンテキスト サイズ、およびより大きなモデル用のスペースのバランスが取れています。

RTX 3060 から RTX 5090 までのカードは正常に動作します。最適なフィット感は依存します
RAM、CPU、冷却、およびインストールするモデル ファイル。 GPU プリセットを使用する
ハードウェアに合わせた Evi のサイドバー。

CPU のみで実行する (「CPU … GB RAM」プリセット) は緊急フォールバックです。
Evi は適切なグラフィック カードがなくても引き続き使用できますが、応答ははるかに遅くなります。
できる限り本物の GPU を使用することをおすすめします。

--------------------------------------------------------------------------------
  完全にポータブル - インターネットが必要なときに
--------------------------------------------------------------------------------

Evi はポータブルであるように構築されています。フォルダー全体を別のドライブ、USB スティック、
または PC を選択し、そこで起動します。チャット、思い出、設定、セキュリティ データ、および
アクティベーション ファイルは通常、そのフォルダー内にあります (アクティベーションは
フォルダー パスではなく PC)。

インターネットは主に、ダウンロードすることを選択したもの (モデル ファイル、オプション) のために使用されます。
音声やエキストラ、アップデート）、および Web 検索、メール、
ストリーミング、またはネットワーク アクセスを許可するときの天気。

コアチャット、メモリ、メモ、タイマー、許可されたフォルダー内のファイルツール、ローカル
音声認識、ローカル音声出力、およびプライバシー ロックは、
会話をクラウド サービスに送信せずに、コンピュータに送信できます。オプションのオンライン
機能は、ネットワーク アクセスがオンで使用されている場合にのみ実行されます。

--------------------------------------------------------------------------------
  セキュリティ: PIN と音声によるロック解除 (コピー防止)
--------------------------------------------------------------------------------

Evi は、デバイス上でセキュリティ PIN とオプションの音声登録を使用できます。

音声ロック解除を使用すると、試行のたびに短いランダムな単語のセットが要求されることがあります。
それは毎回変わります。永遠に 1 つの決まったフレーズがあるわけではありません。それは簡単なことを妨げます
自分の声の古い録音を 1 つだけ再生するというトリック。 PIN はそのまま残ります
独立した防御ライン。

これは、カジュアルな虐待を阻止することを目的としています。消費者向け製品は約束できません
あらゆる攻撃に対する完璧な防御（たとえば、PIN とパスワードの両方を知っている誰か）
洗練された声真似）。 PIN を非公開にして、セットアップを完了してください。
アプリがガイドします。

--------------------------------------------------------------------------------
  トラブルシューティング (短いチェックリスト)
--------------------------------------------------------------------------------

• AI の応答なし / モデルのエラー
  → モデル\llm\ に少なくとも 1 つの適切な .gguf ? models\llm\README.txt を参照してください。
  → GPU パスが失敗した場合は、より小さいモデルまたはサイドバーの CPU プリセットを試してください。

• マイクなし
  → Windows サウンド設定: マイクをテストします。 Evi で、以下のデバイスを選択します。
    マイクと保存。

• 音声出力なし
  → 使用する言語の音声ファイルがpiper\フォルダーに存在することを確認し、
    サイドバーで音声を選択します。

• ウェブまたは YouTube が失敗する
  → オンライン機能が必要な場合は、サイドバーの「すべてのトラフィックをブロック」をオフにします。
  → YouTube を再生するには、通常、VLC Portable とネットワーク アクセスが必要です。

• チートシートの編集後
  → Evi を再起動して、すべてのテキスト ヘルパーが変更を確実に認識できるようにします。

1) オペレーティングシステム
  • Evi は Windows 10 および 11 のみに対応します。他の OS バージョンはサポートされていません。
  • アプリがまったく起動しない場合は、パッケージを完全に解凍し、ウイルス対策をチェックしてください。
    ブロックされた実行可能ファイルのログを取得し、まれな問題が発生しない短いインストール パスを試してください。
    Unicode 専用の文字。

2) フォルダー パス、ウイルス対策、移植性
  • スキャンによって危険な場合は、セキュリティ ソフトウェアに Evi ルート フォルダーの除外を追加します。
    起動が非常に遅い、またはダウンロード中にファイルがロックされる。
  • USB スティック: USB 3 + NTFS を推奨します。メディアが非常に遅いため、大きなモデルは苦痛になります。

3) 言語モデル (.gguf)
  • 症状: 応答なし、モデルエラー、ロードの即時失敗。
  • 修正: models\llm\ に完全な .gguf (0 バイトではない) が含まれていることを確認します。再ダウンロード
必要に応じて; models\llm\README.txt を使用してモデル サイズを VRAM に一致させます。不明な場合は、
    テスト済みの 1 つの q4_k_m ファイルをフォルダー内に保持します。
  • 複数の .gguf ファイル: Evi は、VRAM に適合する最大のものを選択できます。
    競合が疑われる場合は、1 つのファイルのみを使用してテストしてください。

4) GPU、CUDA、ドライバー、メモリ不足
  • NVIDIA ドライバーを更新します。ラップトップでは、ディスクリート GPU / パフォーマンスを強制します
    Windows が許可する Evi のモード。
  • OOM または GPU のクラッシュが発生した場合は、より小さいチェックポイントを使用し、他の GPU アプリを閉じます。
    温度を下げるか、CPU プリセットに切り替えます (低速ですが安定しています)。

5) CPU専用モード
  • 遅いことが予想されます。十分な空きシステム RAM。重いバックグラウンドタスクを閉じます。
    長時間のセッションでは電源プラン「高パフォーマンス」を使用してください。

6) アクティベーション
  • キーを慎重に貼り付けます (余分なスペースは入れないでください)。キーは 1 つのマシン ID に関連付けられます。
  • ハードウェアの大幅な変更後は、交換キーが必要になる場合があります。サポートをご利用ください。
    詳細を記載したアドレスを入力してください。

7) マイク
  • Windows プライバシー → マイク → デスクトップ アプリを許可します。
  • 正しいデフォルトの録音デバイスを設定します。 Evi では同じデバイスを選択して保存します。
  • Bluetooth ヘッドセットでは遅延が増加する可能性があります。不明な場合は、USB マイクを使用してテストしてください。
  • 別のアプリが排他モードを保持している可能性があります。会議ソフトウェアを一時的に閉じます。

8) 音声認識 (ローカル)
  • 認識が完了しない場合は、models\whisper の下にあるモデル ファイルが
    ビルドが必要な場合は、最初のモデル取得時にネットワークを許可します。

9) 音声出力（Piper / TTS）
  •piper\ に UI 言語の音声パックが含まれていることを確認します。声を選ぶ
    サイドバー。 Windows のボリューム ミキサーを確認してください — Evi はアプリごとにミュートされている可能性があります。

10) ネットワークと「すべてのトラフィックをブロック」
  • Web、メールヘルパー、天気予報、またはダウンロードのブロックをオフにします。
  • 企業プロキシまたは VPN では、ホワイトリストに関して IT 部門の支援が必要な場合があります。

11) YouTube とメディア
  • バンドルされている VLC ポータブル レイアウトを維持します。ネットワークアクセスを有効にする。読む
    パスが移動された場合は、Portable\VLCPortable のガイダンス。

12) UI言語とチートシート
  • チートシートは言語ファイルごとに cheatsheets\ にあります。cheatsheets\README.txt を参照してください。
  • 編集内容を UTF-8 として保存します。編集後に Evi を再起動します。

13) セキュリティ(PIN/音声ロック解除)
  • 静かな環境を使用してください。マイクゲインを再確認します。 PIN を決して共有しないでください。
  • ランダムな単語のプロンプトは意図的なものです。古い音声録音はロック解除されるべきではありません。

14) パフォーマンス、フリーズ、クラッシュ
  • モデル サイズを縮小し、冷却を改善し、ツリー内の crash.log を読み取ります。
    build はそれを作成し、失敗する前に行った最後の変更をロールバックします。

15) サポートに連絡する場合
  • マシン ID、Windows バージョン、GPU モデル + VRAM + ドライバー (.gguf) を含めます。
    使用するものと、正確なエラー テキスト (秘密キーではありません)。
