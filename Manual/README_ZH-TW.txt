------------------------------------------------------------------------------------------------
  作業系統
------------------------------------------------------------------------------------------------

Evi Portable 僅適用於 Microsoft Windows 10 和 Windows 11。它不是
Windows 7/8、macOS、Linux 或行動作業系統支援。

------------------------------------------------------------------------------------------------
  第一次開始—從下載到執行 Evi
------------------------------------------------------------------------------------------------

1) 安裝位置
   將整個 Evi 資料夾複製或解壓縮到您控制的位置 - 例如
   您的桌面、文件或 USB 記憶棒。保持資料夾結構為
   已交付（請勿刪除並非您自己建立的子資料夾）。

2）語言模型（必填）
   Evi 需要在該資料夾中放置一個主 AI 模型檔案（副檔名 .gguf）：
   模型\llm\
   有關大小建議，請參閱 models\llm\README.txt。文件到了之後，
   啟動埃維；它會在可能的情況下自動偵測模型。

3) 首次啟動——激活
   當您第一次啟動 Evi 時，會出現一個啟動視窗。
   • 您的PC 顯示機器ID（對於該硬體來說是唯一的）。
   • 複製機器 ID（按鈕：「複製機器 ID」）並透過電子郵件傳送至：
     Brielbeck@hotmail.de
   • 您將收到只能在同一台電腦上使用的序號
     （金鑰與您的硬體綁定）。
   • 將金鑰貼到啟動視窗中並選擇啟動。

   如果您變更主要硬體或以變更 ID 的方式移動磁碟，
   您可能需要一把新鑰匙 - 請聯絡同一地址。

4) 啟動後
   主視窗打開。在左側選擇語言、麥克風和語音
   設定面板。隨時說“打開備忘單”以獲取命令示例
   （請參閱 cheatsheets\en.txt 和其他語言的姊妹檔）。

5) 選用：從 Windows 開啟本手冊
   在檔案總管中，前往 Evi 資料夾，然後前往 Manual\README.txt，然後將其開啟
   使用記事本或任何文字檢視器。

------------------------------------------------------------------------------------------------
  介面語言（GUI）
------------------------------------------------------------------------------------------------

使用者介面遵循您在頂部語言下選擇的語言
左側設定面板的（側邊欄中的第一個區塊）。當您選擇一個
語言、選單、按鈕、標籤和大多數螢幕文字都會切換到該語言
存在翻譯的語言。您可以隨時更改；只有
措辭發生變化，而不是佈局發生變化。

提示：cheatsheets\ 資料夾中的命令備忘單按以下順序列出
語言代碼（en.txt、de.txt、...）。有關詳細信息，請參閱cheatsheets\README.txt。
此 Manual\ 資料夾是較長的產品概述；備忘單是
快速“說什麼”列表。

------------------------------------------------------------------------------------------------
  顯示卡 (GPU) — 快速指南
------------------------------------------------------------------------------------------------

一個強大的中間選擇是具有 12 GB 視訊的 NVIDIA GeForce RTX 4070 Ti
記憶體：速度、上下文大小和較大模型空間的良好平衡。

從 RTX 3060 到 RTX 5090 左右的卡都可以很好地工作；最合適的取決於
關於 RAM、CPU、冷卻以及您安裝的型號檔案。使用 GPU 預設
Evi 的側邊欄可搭配您的硬體。

僅在 CPU 上運作（「CPU … GB RAM」預設）是緊急後備方案：
Evi 在沒有合適的顯示卡的情況下仍然可以使用，但反應速度要慢得多。
盡可能選擇真正的 GPU。

------------------------------------------------------------------------------------------------
  完全便攜－當您需要網路時
------------------------------------------------------------------------------------------------

Evi 設計為可攜式：將整個資料夾複製到另一個磁碟機、USB 記憶棒、
或 PC，然後在那裡啟動它。聊天、記憶、設定、安全資料和您的
激活檔案通常保留在該資料夾中（激活與
PC，不是資料夾路徑）。

互聯網主要是供您選擇下載的東西（模型文件，可選
語音或附加功能、更新）以及選購功能，例如網路搜尋、郵件、
串流媒體或天氣（當您允許網路存取時）。

核心聊天、記憶體、筆記、計時器、允許資料夾內的檔案工具、本地
語音辨識、本地語音輸出和隱私鎖定運行在您的
機器，無需將對話傳送到雲端服務。可選在線
功能僅在網路存取開啟並且您使用它們時運行。

------------------------------------------------------------------------------------------------
  安全性：PIN 碼和語音解鎖（防複製）
------------------------------------------------------------------------------------------------

Evi 可以在您的裝置上使用安全 PIN 碼和可選的語音註冊。

使用語音解鎖時，每次嘗試都會要求輸入一組簡短的隨機單字
每次都會改變—沒有一個永遠固定的短語。這阻礙了簡單的
重播您的舊錄音的技巧。您的 PIN 碼仍然是
單獨的防線。

這是為了阻止隨意濫用行為；沒有任何消費品可以承諾
完美抵禦每次攻擊（例如，某人同時擁有您的 PIN 碼和
複雜的聲音模仿）。將您的 PIN 碼保密並完成設置
應用程式指導您。

------------------------------------------------------------------------------------------------
  故障排除（簡短清單）
------------------------------------------------------------------------------------------------

• 無人工智慧答案/模型錯誤
  → models\llm\ 中至少有一個合適的 .gguf ？請參閱 models\llm\README.txt。
  → 如果 GPU 路徑失敗，請嘗試較小的模型或側邊欄中的 CPU 預設。

• 無麥克風
  → Windows 聲音設定：測試麥克風。在 Evi 中，選擇下面的設備
    麥克風並保存。

• 無語音輸出
  → 檢查 Piper\ 資料夾中是否存在適合您語言的語音文件，然後
    在側欄中選擇一個聲音。

• Web 或 YouTube 失敗
  → 如果您需要線上功能，請關閉側邊欄中的「封鎖所有流量」。
  → 對於 YouTube 播放，通常需要 VLC Portable 和網路存取。

• 編輯備忘錄後
  → 重新啟動 Evi，以便所有文字助理都能可靠地取得變更。

------------------------------------------------------------------------------------------------
  故障排除（擴充參考）
------------------------------------------------------------------------------------------------

快速檢查（始終是第一位）
  • Windows 10 或11 64 位元、更新、重大變更後重新啟動？
  • Evi 從完全解壓縮的資料夾運作－而不是從存檔內部運作？
  • 避免使用雲端同步資料夾（OneDrive 等）作為即時資料資料夾，同時
    Evi 運行 — 使用本機路徑，例如 C:\EviPortable 或 D:\Tools\Evi
    可能的。
  • 有足夠的可用磁碟空間用於模型、聊天和更新嗎？
  • 更改模型、聲音或備忘單後：完全退出 Evi 並開始
    再次。

1）作業系統
  • Evi 僅適用於Windows 10 和11；不支援其他作業系統版本。
  • 如果應用程式根本無法啟動，請完全解壓縮軟體包，檢查防毒軟體
    記錄被阻止的可執行文件，並嘗試較短的安裝路徑而不罕見
    僅限 Unicode 字元。

2) 資料夾路徑、防毒、可移植性
  • 如果掃描結果為您的 Evi 根資料夾，請在安全軟體中新增排除項
    啟動速度非常慢或在下載過程中鎖定檔案。
  • U 碟：首選 USB 3 + NTFS；非常緩慢的媒體使大型模型變得痛苦。

3）語言模型（.gguf）
  • 症狀：無回覆、模型錯誤、即時載入失敗。
  • 修復：驗證models\llm\ 包含完整的.gguf（不是0 位元組）；重新下載
如果需要的話；使用 models\llm\README.txt 將模型大小與 VRAM 配對；如果不確定，
    在資料夾中保留一個經過測試的 q4_k_m 檔案。
  • 多個.gguf 檔案：Evi 可以選擇適合您 VRAM 的最大檔案 — 如果您
    懷疑有衝突，僅使用一個文件進行測試。

4) GPU、CUDA、驅動程式、記憶體不足
  • 更新NVIDIA驅動程式；在筆記型電腦上，強制使用獨立 GPU/效能
    Windows 允許的 Evi 模式。
  • 如果您發現 OOM 或 GPU 崩潰，請使用較小的檢查點，關閉其他 GPU 應用程序，
    降低溫度，或切換到 CPU 預設（較慢但更穩定）。

5) 僅CPU模式
  • 預計會很慢；釋放足夠的系統 RAM；關閉繁重的後台任務；
    在長時間會話期間使用「高效能」電源計劃。

6) 激活
  • 小心地貼上按鍵（不要有多餘的空格）；金鑰與一個機器 ID 綁定。
  • 在進行重大硬體變更後，您可能需要更換金鑰 - 請使用支持
    地址和您的詳細資料。

7) 麥克風
  • Windows 隱私→ 麥克風→ 允許桌面應用程式。
  • 設定正確的預設錄音設備；在 Evi 中選擇相同的設備並儲存。
  • 藍牙耳機會增加延遲－如果不確定，請使用USB 麥克風進行測試。
  • 另一個應用程式可能會保持獨佔模式—暫時關閉會議軟體。

8) 語音辨識（本地）
  • 如果識別從未完成，請確保 models\whisper 下的模型文件
    完好無損，並且如果您的構建需要，則允許網路首次獲取模型。

9) 語音輸出（Piper/TTS）
  • 確認 Piper\ 包含適合您的 UI 語言的語音包；選擇一個聲音
    側邊欄；檢查 Windows 音量混合器 — Evi 可能會根據應用程式靜音。

10) 網路和“阻止所有流量”
  • 關閉網路、郵件助理、天氣或下載的封鎖。
  • 公司代理商或VPN 可能需要IT 協助來決定白名單。

11) YouTube 和媒體
  • 保留捆綁的VLC Portable 佈局；啟用網路存取；讀取
    Portable\VLCPortable 指導（如果路徑已移動）。

12) UI 語言和備忘單
  • 備忘單是cheatsheets\ 下的每種語言檔案－請參閱cheatsheets\README.txt。
  • 將編輯儲存為UTF-8；編輯後重新啟動 Evi。

13) 安全性（PIN/語音解鎖）
  • 使用安靜的環境；重新檢查麥克風增益；切勿分享您的 PIN 碼。
  • 隨機單字提示是有意為之的－舊的錄音不應解鎖。

14) 性能、凍結、崩潰
  • 縮小模型尺寸，改善冷卻，閱讀樹中的任何崩潰日誌（如果您的情況如此）
    build 建立一個，並回滾失敗之前所做的最後更改。

15) 聯繫支援人員時
  • 包含機器 ID、Windows 版本、GPU 型號 + VRAM + 驅動程序，其中 .gguf
    您使用的，以及確切的錯誤文字（不是秘密密鑰）。
