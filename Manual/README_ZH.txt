也可以将您自己的较长指南或翻译放在这里（例如 EN_User_Manual.pdf，

------------------------------------------------------------------------------------------------
  操作系统
------------------------------------------------------------------------------------------------

Evi Portable 仅适用于 Microsoft Windows 10 和 Windows 11。它不是
Windows 7/8、macOS、Linux 或移动操作系统支持。

------------------------------------------------------------------------------------------------
  第一次开始——从下载到运行 Evi
------------------------------------------------------------------------------------------------

1) 安装位置
   将整个 Evi 文件夹复制或解压缩到您控制的位置 - 例如
   您的桌面、文档或 USB 记忆棒。保持文件夹结构为
   已交付（请勿删除并非您自己创建的子文件夹）。

2）语言模型（必填）
   Evi 需要在该文件夹中放置一个主 AI 模型文件（扩展名 .gguf）：
   模型\llm\
   有关大小建议，请参阅 models\llm\README.txt。文件到了之后，
   启动埃维；它会在可能的情况下自动检测模型。

3) 首次启动——激活
   当您第一次启动 Evi 时，会出现一个激活窗口。
   • 您的PC 显示机器ID（对于该硬件来说是唯一的）。
   • 复制机器 ID（按钮：“复制机器 ID”）并通过电子邮件发送至：
     Brielbeck@hotmail.de
   • 您将收到只能在同一台电脑上使用的序列号
     （密钥与您的硬件绑定）。
   • 将密钥粘贴到激活窗口中并选择激活。

   如果您更改主要硬件或以更改 ID 的方式移动磁盘，
   您可能需要一把新钥匙 - 请联系同一地址。

4) 激活后
   主窗口打开。在左侧选择语言、麦克风和语音
   设置面板。随时说“打开备忘单”以获取命令示例
   （请参阅 cheatsheets\en.txt 和其他语言的姊妹文件）。

5) 可选：从 Windows 打开本手册
   在文件资源管理器中，转到 Evi 文件夹，然后转到 Manual\README.txt，然后将其打开
   使用记事本或任何文本查看器。

------------------------------------------------------------------------------------------------
  界面语言（GUI）
------------------------------------------------------------------------------------------------

用户界面遵循您在顶部语言下选择的语言
左侧设置面板的（侧边栏中的第一个块）。当您选择一个
语言、菜单、按钮、标签和大多数屏幕文本都会切换到该语言
存在翻译的语言。您可以随时更改；只有
措辞发生变化，而不是布局发生变化。

提示：cheatsheets\ 文件夹中的命令备忘单按以下顺序列出
语言代码（en.txt、de.txt、...）。有关详细信息，请参阅cheatsheets\README.txt。
此 Manual\ 文件夹是较长的产品概述；备忘单是
快速“说什么”列表。

------------------------------------------------------------------------------------------------
  显卡 (GPU) — 快速指南
------------------------------------------------------------------------------------------------

一个强大的中间选择是具有 12 GB 视频的 NVIDIA GeForce RTX 4070 Ti
内存：速度、上下文大小和较大模型空间的良好平衡。

从 RTX 3060 到 RTX 5090 左右的卡都可以很好地工作；最合适的取决于
关于 RAM、CPU、冷却以及您安装的型号文件。使用 GPU 预设
Evi 的侧边栏可匹配您的硬件。

仅在 CPU 上运行（“CPU … GB RAM”预设）是紧急后备方案：
Evi 在没有合适的显卡的情况下仍然可以使用，但响应速度要慢得多。
尽可能选择真正的 GPU。

------------------------------------------------------------------------------------------------
  完全便携——当您需要互联网时
------------------------------------------------------------------------------------------------

Evi 设计为便携式：将整个文件夹复制到另一个驱动器、USB 记忆棒、
或 PC，然后在那里启动它。聊天、记忆、设置、安全数据和您的
激活文件通常保留在该文件夹中（激活与
PC，不是文件夹路径）。

互联网主要是供您选择下载的东西（模型文件，可选
语音或附加功能、更新）以及可选功能，例如网络搜索、邮件、
流媒体或天气（当您允许网络访问时）。

核心聊天、内存、笔记、计时器、允许文件夹内的文件工具、本地
语音识别、本地语音输出和隐私锁运行在您的
机器，无需将对话发送到云服务。可选在线
功能仅在网络访问打开并且您使用它们时运行。

------------------------------------------------------------------------------------------------
  安全性：PIN 码和语音解锁（防复制）
------------------------------------------------------------------------------------------------

Evi 可以在您的设备上使用安全 PIN 码和可选的语音注册。

使用语音解锁时，每次尝试都会要求输入一组简短的随机单词
每次都会发生变化——没有一个永远固定的短语。这阻碍了简单的
重放您的旧录音的技巧。您的 PIN 码仍然是
单独的防线。

这是为了阻止随意滥用行为；没有任何消费品可以承诺
完美抵御每次攻击（例如，某人同时拥有您的 PIN 码和
复杂的声音模仿）。将您的 PIN 码保密并完成设置
应用程序指导您。

------------------------------------------------------------------------------------------------
  故障排除（简短清单）
------------------------------------------------------------------------------------------------

• 无人工智能答案/模型错误
  → models\llm\ 中至少有一个合适的 .gguf ？请参阅 models\llm\README.txt。
  → 如果 GPU 路径失败，请尝试较小的模型或侧栏中的 CPU 预设。

• 无麦克风
  → Windows 声音设置：测试麦克风。在 Evi 中，选择下面的设备
    麦克风并保存。

• 无语音输出
  → 检查 Piper\ 文件夹中是否存在适合您语言的语音文件，然后
    在侧栏中选择一个声音。

• Web 或 YouTube 失败
  → 如果您需要在线功能，请关闭侧边栏中的“阻止所有流量”。
  → 对于 YouTube 播放，通常需要 VLC Portable 和网络访问。

• 编辑备忘单后
  → 重新启动 Evi，以便所有文本助手都能可靠地获取更改。

------------------------------------------------------------------------------------------------
  故障排除（扩展参考）
------------------------------------------------------------------------------------------------

快速检查（始终是第一位）
  • Windows 10 或11 64 位、更新、重大更改后重新启动？
  • Evi 从完全解压的文件夹运行——而不是从存档内部运行？
  • 避免使用云同步文件夹（OneDrive 等）作为实时数据文件夹，同时
    Evi 运行 — 使用本地路径，例如 C:\EviPortable 或 D:\Tools\Evi
    可能的。
  • 有足够的可用磁盘空间用于模型、聊天和更新吗？
  • 更改模型、声音或备忘单后：完全退出 Evi 并开始
    再次。

1）操作系统
  • Evi 仅适用于Windows 10 和11；不支持其他操作系统版本。
  • 如果应用程序根本无法启动，请完全解压软件包，检查防病毒软件
    记录被阻止的可执行文件，并尝试较短的安装路径而不罕见
    仅限 Unicode 字符。

2) 文件夹路径、防病毒、可移植性
  • 如果扫描结果为您的 Evi 根文件夹，请在安全软件中添加排除项
    启动速度非常慢或在下载过程中锁定文件。
  • U 盘：首选 USB 3 + NTFS；非常慢的媒体使大型模型变得痛苦。

3）语言模型（.gguf）
  • 症状：无回复、模型错误、即时加载失败。
  • 修复：验证models\llm\ 包含完整的.gguf（不是0 字节）；重新下载
如果需要的话；使用 models\llm\README.txt 将模型大小与 VRAM 匹配；如果不确定，
    在文件夹中保留一个经过测试的 q4_k_m 文件。
  • 多个.gguf 文件：Evi 可以选择适合您 VRAM 的最大文件 — 如果您
    怀疑存在冲突，仅使用一个文件进行测试。

4) GPU、CUDA、驱动程序、内存不足
  • 更新NVIDIA驱动程序；在笔记本电脑上，强制使用独立 GPU/性能
    Windows 允许的 Evi 模式。
  • 如果您发现 OOM 或 GPU 崩溃，请使用较小的检查点，关闭其他 GPU 应用程序，
    降低温度，或切换到 CPU 预设（较慢但更稳定）。

5) 仅CPU模式
  • 预计会很慢；释放足够的系统 RAM；关闭繁重的后台任务；
    在长时间会话期间使用“高性能”电源计划。

6) 激活
  • 小心地粘贴按键（不要有多余的空格）；密钥与一个机器 ID 绑定。
  • 在进行重大硬件更改后，您可能需要更换密钥 - 请使用支持
    地址和您的详细信息。

7) 麦克风
  • Windows 隐私→ 麦克风→ 允许桌面应用程序。
  • 设置正确的默认录音设备；在 Evi 中选择相同的设备并保存。
  • 蓝牙耳机会增加延迟——如果不确定，请使用USB 麦克风进行测试。
  • 另一个应用程序可能会保持独占模式——暂时关闭会议软件。

8) 语音识别（本地）
  • 如果识别从未完成，请确保 models\whisper 下的模型文件
    完好无损，并且如果您的构建需要，则允许网络首次获取模型。

9) 语音输出（Piper/TTS）
  • 确认 Piper\ 包含适合您的 UI 语言的语音包；选择一个声音
    侧边栏；检查 Windows 音量混合器 — Evi 可能会根据应用程序静音。

10) 网络和“阻止所有流量”
  • 关闭对网络、邮件助手、天气或下载的阻止。
  • 公司代理或VPN 可能需要IT 帮助来确定白名单。

11) YouTube 和媒体
  • 保留捆绑的VLC Portable 布局；启用网络访问；读
    Portable\VLCPortable 指导（如果路径已移动）。

12) UI 语言和备忘单
  • 备忘单是cheatsheets\ 下的每种语言文件——请参阅cheatsheets\README.txt。
  • 将编辑保存为UTF-8；编辑后重新启动 Evi。

13) 安全（PIN/语音解锁）
  • 使用安静的环境；重新检查麦克风增益；切勿分享您的 PIN 码。
  • 随机单词提示是有意为之的——旧的录音不应解锁。

14) 性能、冻结、崩溃
  • 缩小模型尺寸，改善冷却，阅读树中的任何崩溃日志（如果您的情况如此）
    build 创建一个，并回滚失败之前所做的最后更改。

15) 联系支持人员时
  • 包含机器 ID、Windows 版本、GPU 型号 + VRAM + 驱动程序，其中 .gguf
    您使用的，以及确切的错误文本（不是秘密密钥）。

完整的故障排除参考（此语言）：Manual\FEHLERSUCHE_ZH.txt
