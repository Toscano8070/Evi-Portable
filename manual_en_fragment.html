Evi Portable — User manual (overview)

This overview matches Manual\README.txt in your build. It is optional for Evi to run; read or print it for setup and troubleshooting.

Operating system

Evi Portable is for Microsoft Windows 10 and Windows 11 only. It is not supported on Windows 7/8, macOS, Linux, or mobile operating systems.

First start — from download to running Evi

  1. Install location. Copy or extract the whole Evi folder somewhere you control — Desktop, Documents, or a USB stick. Keep the folder structure as delivered (do not delete subfolders you did not create yourself).
  2. Download the program. Use Get Evi Portable or the download card on this page. That archive is the application; it does not include the main LLM file.
  3. Language model (required). Evi needs one main AI model file (.gguf) in models\llm\. On this site, click Download AI model (GGUF) in the hero section when the publisher has attached the link — that is the recommended package for this build. Until then, the button jumps here; you may use any compatible .gguf from another source. See models\llm\README.txt in your folder for size and preset hints.

Right model, right place: After you click Download AI model (GGUF) and obtain the file, place exactly one suitable .gguf in models\llm\, then start Evi; it detects the model when possible.

Download AI model (GGUF)

⚠️ High-end NVIDIA GPUs only — e.g. GeForce RTX 5090 class

The 14B instruct GGUF is qwen2.5-14b-instruct-q4_k_m.gguf — click the filename to start the LLM model download (that archive contains this file). For top-tier cards with plenty of VRAM (think RTX 5090 and similar) only. Do not attempt to use this checkpoint on anything less — you risk out-of-memory errors, unusable speed, or crashes. If your GPU is below that, use the standard LLM download on the Download section and a smaller model per models\llm\README.txt, not this 14B file.

  1. First launch — activation. When you start Evi the first time, an activation window appears. Your PC shows a Machine ID. Use Copy Machine ID and e-mail it to Brielbeck@hotmail.de. You receive a serial key that works only on that PC. Paste the key and choose Activate. If you change major hardware in a way that changes the ID, contact the same address for a replacement key.
  2. After activation. The main window opens. Choose language, microphone, and voice in the left settings panel. Say “open the cheat sheet” anytime for command examples (cheatsheets\en.txt and sister files; see cheatsheets\README.txt).
  3. Optional: open this manual from Windows. In File Explorer open Manual\README.txt with Notepad or any text viewer.

Interface language (GUI)

The UI follows the language you choose under Language at the top of the left settings panel. After you select a language, menus, buttons, and most labels switch where translations exist. Cheat sheets are per language code (en.txt, de.txt, …). This manual is the longer overview; cheat sheets are the quick “what to say” list.

Graphics card (GPU) — quick guide

A strong middle choice is an NVIDIA GeForce RTX 4070 Ti with 12 GB VRAM: good balance of speed, context size, and headroom. Cards from about RTX 3060 through RTX 5090 can work well; the best fit depends on RAM, CPU, cooling, and which model file you install. Match the GPU preset in Evi’s sidebar to your hardware. CPU-only presets are a fallback: usable but much slower — prefer a real GPU when you can.

Fully portable — when you need the Internet

Copy the whole folder to another drive, USB stick, or PC and start it there. Chats, memories, settings, security data, and your activation file normally stay inside that folder (activation is tied to the PC, not the folder path). The Internet is mainly for downloads (models, optional voices, updates) and for optional features you enable (web, mail, streaming, weather). Core chat, memory, local speech, local TTS, and the privacy lock run on your machine without sending conversations to a cloud service.

Security: PIN and voice unlock (anti-copy)

Evi can use a security PIN and optional voice enrolment. With voice unlock, each attempt can ask for a short set of random words that changes every time — not one fixed phrase forever — which blocks replaying a single old recording. Your PIN remains a separate line of defence. This targets casual abuse; keep your PIN private and complete setup as the app guides you.

Troubleshooting

Quick list below, then detailed topics. German printable reference: Manual\FEHLERSUCHE_DE.txt inside your Evi folder.

  • No AI / model — At least one suitable .gguf in models\llm\? See models\llm\README.txt. Try a smaller model or a CPU sidebar preset if GPU fails.
  • No microphone — Windows privacy + sound test; pick device in Evi → Microphone → Save.
  • No voice — Voices under piper\; select voice in sidebar; check volume mixer.
  • Web / YouTube — Disable “Block all traffic” if you want online tools; YouTube usually needs network + VLC Portable.
  • Cheat sheets — After edits, restart Evi; UTF-8; see cheatsheets\README.txt.

Always check first

Windows 10 or 11 64-bit, updated; run from a fully extracted folder (not half-unzipped). Prefer a local disk path (avoid cloud-sync folders like OneDrive for live data). Enough free space. Fully exit and restart Evi after model or cheat-sheet changes.

1. OS and paths

Other operating systems are unsupported. Very long paths or aggressive antivirus can block or slow the app; add a folder exclusion if needed. USB: use fast media and NTFS.

2. Model file (.gguf)

Incomplete downloads (0 Byte or cut-off files) break loading. Re-download. Multiple .gguf files can coexist; Evi may pick the largest that fits your VRAM — see models\llm\README.txt. If confused, test with only one known-good file.

3. GPU, drivers, memory

Update NVIDIA drivers. On laptops, prefer discrete / performance GPU for Evi. Out of memory: smaller model, close other GPU apps, improve cooling, or use CPU preset. TDR / driver reset: reduce load or temperature; avoid unstable overclocks.

4. CPU-only

Slow replies are normal. Free RAM; “High performance” power plan; close heavy background tasks.

5. Activation

Paste the serial carefully (no spaces). Keys bind to one Machine ID. After major hardware change you may need a replacement key from support with proof of purchase.

6. Microphone and speech

Settings → Privacy → Microphone → allow desktop apps. Correct default recording device. Bluetooth adds latency — try USB. Exclusive mode in other apps (meetings) can block the mic.

7. Piper / TTS

Language-appropriate files must exist under piper\. Pick a matching voice; ensure Evi is not muted in the system mixer.

8. Network

Disable global block in Evi when using web/mail/weather/downloads. Corporate VPN/proxy may need IT allowlisting.

9. Media / YouTube

Do not delete the portable VLC layout; allow network access for streaming helpers.

10. Security (PIN / voice)

Use a quiet room; random-word prompts are anti-replay. Never share your PIN.

11. Crashes / slowness

Check thermal throttling, disk space, and any crash.log your build writes. Roll back the last change before failures.

12. Contacting support

Send Machine ID, Windows version, GPU + VRAM + driver, which .gguf, steps to reproduce, error text (never paste secret keys in public screenshots).

Support — serial keys and Machine ID

E-mail: Brielbeck@hotmail.de. Always include your Machine ID from the activation screen when requesting a serial key or a replacement after hardware changes.
